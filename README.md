# Social Gravity

**Social Gravity** is a unified social media posting application that allows users to manage and schedule content across multiple platforms from a single dashboard.

## Project Structure

The project is divided into two main components:

- **[Backend](backend/README.md)**: A Spring Boot application handling API requests, authentication, and database interactions.
- **[Frontend](frontend/README.md)**: An Angular application providing the user interface for managing accounts and posts.

## Technology Stack

- **Backend**: Java 17, Spring Boot 3, PostgreSQL, Spring Security (JWT), ScribeJava (OAuth).
- **Frontend**: Angular 17, TailwindCSS, TypeScript.
- **Database**: PostgreSQL.
